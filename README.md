# steganography-toolkit
# 🔐 Multi-Format Steganography Toolkit

A web-based and CLI tool for hiding and extracting secret data using steganography techniques with encryption.

---

## 🚀 Features

- 🖼 Image Steganography (LSB Encoding)
- 🔐 AES Encryption
- 🔍 Steganography Detection
- 📝 Text Steganography (Zero-width Unicode)
- 📊 Capacity Analysis
- 📈 Histogram Visualization
- 🌐 Streamlit Web Interface

---

## 🛠 Tech Stack

- Python
- Streamlit
- Pillow (Image Processing)
- Cryptography (AES Encryption)
- NumPy & Matplotlib (Analysis)
